字段解释:
```
CI:
screen_size //屏幕尺寸
color_depth //屏幕色深
app_code //appCode
app_name //appName
cpu //获取CPU信息
platform //获取platform
network //获取网络连接类型
language //获取系统语言
timezone //获取时区
flash_ver //获取flash版本
ua //获取浏览器UA
```
```
PI:
pagename //获取页面title
referrer //获取document.referer
is_homepage //获取当前页是否为默认页
dom_count //获取DOM节点数量
iframe_count //获取页面iframe数
url //获取当前页面的url
env //获取当前网站环境 | dev,test,pre,pro
```
```
UI:
phpsid //获取phpsid
ssid //获取唯一用户标记 5年
is_new //7天是否访问过此页面
uid //用户id
shop_id //店铺id
produce_id //商品id
group_id //圈子id
topic_id //话题id
channel //频道id
```
```

active_no //获取active_no属性
page_id //获取页面id

```
```

sdate //获取页面服务器时间
load_time //页面加载结束时间onload
```
