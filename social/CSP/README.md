## 开发相关

* `git clone http://gitlab.intra.gomeplus.com/gomeplusFED/CSP.git`

* `cd CSP`

* `npm install -d`

* `npm run localdev` 本地开发，连接AMP平台

* `npm run localtest` 本地开发，连接后台人员本机

* `npm run formatall` 格式化所有文件
